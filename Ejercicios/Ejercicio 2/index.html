World Wide Web

En informatica, la World Wide Web (WWW) o red informatica mundial comunmente conocida como la web, es un sistema de distribucion de documentos de hipertexto o hipermedios interconectados y accesibles via Internet. Con un Navegador web, un usuario visualiza sitios web compuestos de paginas web que pueden contener textos, imagenes, videos u otros contenidos multimedia, y navegar a traves de paginas web usando hiperenlaces

Historia

La web se desarrollo entre marzo de 1989 y diciembre de 1990 por el ingles Tim Berner-Lee con la ayuda del belga Robert Cailliau mientras trabajaban en el CERN en Ginebra, Suiza, y publicado en 1992.
